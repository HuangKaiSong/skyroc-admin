# Skyroc Native UI

A React Native component library built with NativeWind and tailwind-variants, designed for cross-platform mobile development.

## Features

- **NativeWind v4** — Tailwind CSS for React Native
- **tailwind-variants** — Type-safe variant API with slots
- **Cross-platform tokens** — Shared design language with Web components
- **Expo ready** — Works with Expo Router and Metro

## Installation

```bash
pnpm add @skyroc/native-ui @skyroc/native-theme
```

Make sure you have NativeWind configured in your project:

```bash
pnpm add nativewind tailwindcss
```

## Quick Start

```tsx
import { Button, Typography } from '@skyroc/native-ui';

const App = () => {
  return (
    <>
      <Typography size="2xl" weight="bold">Hello Skyroc</Typography>
      <Button color="primary" onPress={() => console.log('pressed')}>
        Get Started
      </Button>
    </>
  );
};
```

## Tailwind Configuration

Extend your `tailwind.config.ts` with the native theme preset:

```ts
import { nativewindPreset } from '@skyroc/native-theme/preset';

export default {
  content: ['./src/**/*.{ts,tsx}'],
  presets: [nativewindPreset],
};
```
