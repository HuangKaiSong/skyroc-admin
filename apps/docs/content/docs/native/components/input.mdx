# Input

A text input component with multiple visual variants, sizes, and support for leading/trailing slots.

<ComponentPreview component="input" />

## Import

```tsx
import { Input } from '@skyroc/native-ui';
```

## Usage

```tsx
<Input placeholder="Enter your name" variant="outline" />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'outline' \| 'filled' \| 'underline'` | `'outline'` | Visual style variant |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Input size |
| `disabled` | `boolean` | `false` | Disable input |
| `error` | `boolean` | `false` | Show error state |
| `leading` | `ReactNode` | — | Content before the input |
| `trailing` | `ReactNode` | — | Content after the input |
| `classNames` | `{ root?: string; input?: string }` | — | Override slot class names |

## Variants

### Outline (default)

```tsx
<Input variant="outline" placeholder="Outline input" />
```

### Filled

```tsx
<Input variant="filled" placeholder="Filled input" />
```

### Underline

```tsx
<Input variant="underline" placeholder="Underline input" />
```
