# Text (Typography)

A typography component for displaying text with consistent sizing, weight, and color.

<ComponentPreview component="text" />

## Import

```tsx
import { Typography } from '@skyroc/native-ui';
```

## Usage

```tsx
<Typography size="2xl" weight="bold" color="primary">
  Hello World
</Typography>
```

## Props

| Prop        | Type                                                              | Default        | Description                   |
| ----------- | ----------------------------------------------------------------- | -------------- | ----------------------------- |
| `size`      | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl' \| '2xl' \| '3xl' \| '4xl'` | `'md'`         | Font size                     |
| `weight`    | `'normal' \| 'medium' \| 'semibold' \| 'bold'`                    | `'normal'`     | Font weight                   |
| `color`     | `'foreground' \| 'muted' \| 'primary' \| ...`                     | `'foreground'` | Text color                    |
| `className` | `string`                                                          | —              | Additional NativeWind classes |
